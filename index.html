About the Project

I've always been fascinated by the potential of cloud computing, and I wanted to challenge myself to build a project that utilized its power. I also felt that the traditional resume format was outdated and lacked interactivity. I wanted to create something that was not only informative but also engaging and visually appealing. .

This project is a portfolio project for Holberton School. GitHub Repository
